# Shift Schedule Planner

A browser-based shift scheduling tool for workforce planning.

## Overview

This project helps generate weekly staff schedules based on configurable workforce rules. It supports employee count setup, work hours, days off, enabled shifts, day-by-day staffing ratios, and shift distribution targets.

## Features

- Configure total employees, working hours, and days off
- Enable or disable up to 8 shifts
- Set staffing ratios by day of week
- Set staffing ratios by shift
- Generate a weekly shift schedule automatically
- View daily staffing summary against target ratios
- Export the generated schedule to Excel
- Save and load settings in the browser

## Use Cases

- Workforce management planning
- Contact center and operations scheduling
- Internal staffing simulations
- Quick what-if staffing experiments

## Tech Stack

- HTML
- CSS
- JavaScript
- SheetJS (XLSX) for Excel export

## Run Locally

Because this is a static single-file app, you can run it directly:

1. Download or clone this repository
2. Open `index.html` in your browser

## Notes

This project is designed as a lightweight front-end scheduling utility and does not require a backend service.
